#define __HAL_RCC_ADC_CONFIG (   __ADC_CLKSOURCE__)     MODIFY_REG(RCC->CCIPR, RCC_CCIPR_ADCSEL, (uint32_t)(__ADC_CLKSOURCE__))

Macro to configure the ADC interface clock.

Parameters
__ADC_CLKSOURCE__specifies the ADC digital interface clock source. This parameter can be one of the following values:
  • RCC_ADCCLKSOURCE_PLLADC PLL "P" (PLLADC) Clock selected as ADC clock
  • RCC_ADCCLKSOURCE_SYSCLK System Clock selected as ADC clock
  • RCC_ADCCLKSOURCE_HSI HSI Clock selected as ADC clock
